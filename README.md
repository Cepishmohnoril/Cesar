# README

This program encrypts and decrypts a 100 character text using Cesar encryption algorithm.

Usage:
* Build
* Encryption

	$ ./cesar <key_int> e

* Decryption

	$ ./cesar <key_int> d

Sample:

	$ ./cesar 123 e
	Please enter a text[100]:
	Hello World
	Axeeh Phkew

	$ ./cesar 123 d
	Please enter a text[100]:
	Axeeh Phkew
	Hello World
